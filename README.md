# Banking-Dashboard-
A clean dashboard showing balance, income, expenses, savings, and recent transactions.

🚀 Features

• Display financial stats
• Show recent transactions
• Clean card‑based UI


🧩 Tech Stack

• HTML
• CSS
• JavaScript


📂 Project Structure

index.html
style.css
script.js


📸 Preview

Shows financial summary and a transaction list.

🧠 How It Works

Financial data is stored in variables and rendered into the dashboard. Transactions are looped and displayed dynamically.
